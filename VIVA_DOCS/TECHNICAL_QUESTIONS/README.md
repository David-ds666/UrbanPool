# Technical Questions Pack

## Purpose
This folder is made for technical viva preparation.

Paste this folder or any file from it into ChatGPT, Gemini, or any AI tool and ask:

- "Answer these questions in simple student language."
- "Ask me these questions one by one."
- "Explain each answer with reference to my code files."
- "Give me short viva answers, not long theory."
- "Convert this into flashcards."

## Best Order To Study
1. `01_FRONTEND_CODE_QUESTIONS.md`
2. `02_BACKEND_API_DATABASE_QUESTIONS.md`
3. `03_FEATURE_FLOW_CODE_MAPPING.md`
4. `04_AI_PROMPTS_TO_USE.md`
