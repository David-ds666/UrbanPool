# 04 - AI Prompts To Use

Use these prompts after pasting any file from this folder into ChatGPT/Gemini.

## Short Answer Prompt

I am preparing for my college viva for a full-stack ride-sharing project called UrbanPool. Use the questions and code references I pasted. Give me short, simple answers in student language. Mention which file or function does the work. Do not give long theory.

## Quiz Mode Prompt

Ask me these questions one by one. Wait for my answer. Then correct me and give the ideal viva answer in 3-5 lines. Focus on technical questions like which file does what, which API is used, and how data flows.

## Code Mapping Prompt

Use this project information and explain "which code does which work." For every feature, tell me the frontend file, backend route/function, database model, and data flow in short points.

## Teacher Follow-Up Prompt

Act like my viva teacher. Ask follow-up questions from my answer. Focus on React, Firebase, Express, MySQL, Sequelize, Socket.io, APIs, routing, authentication, booking flow, wallet, driver tracking, and admin panel.

## Explain One Feature Prompt

Explain this feature in my UrbanPool project:

Feature name: [write feature here]

Answer format:
1. What user does
2. Frontend files involved
3. Backend APIs/functions involved
4. Database models involved
5. Short technical explanation
6. Possible teacher questions

## Convert To Flashcards Prompt

Convert these technical questions into flashcards. Front side should be the teacher's question. Back side should be a short answer with file/function names.

## Weak Areas Prompt

From these questions, identify topics I may be weak in for viva. Make a priority list and teach me only the important parts with examples from my project.

## 2 Minute Explanation Prompt

Give me a 2-minute spoken explanation of the technical architecture of UrbanPool. Include React frontend, Firebase Auth, Express backend, MySQL/Sequelize database, Socket.io real-time features, pricing APIs, wallet, driver portal, and admin panel.

## File-Based Prompt

I will paste one code file from UrbanPool. Explain:
1. What this file is responsible for
2. Important functions/components inside it
3. Which other files it connects with
4. What teacher can ask from this file
5. Short answers for those questions

## API-Based Prompt

Explain all APIs from my backend in viva style. For each API, give:
1. Method and URL
2. What frontend page calls it
3. What database model it uses
4. What response it returns
5. Possible viva question

## Emergency Revision Prompt

I have very little time. From this UrbanPool technical question bank, give me only the top 30 technical questions most likely to be asked by a teacher, with short answers.
