# UrbanPool — Viva Preparation Folder

## How to use this folder
Paste any of these files into ChatGPT / Gemini and ask:
- "Explain this to me in simple language"
- "What questions might a teacher ask about this?"
- "Give me a 2-minute explanation I can say out loud"

## One-sentence project description (say this in viva)
> "UrbanPool is a full-stack ride-sharing web application built with React.js on the frontend and Node.js/Express on the backend, using MySQL for data storage and Firebase for authentication, featuring real-time driver tracking, AI chat support, dynamic pricing, and a complete admin dashboard."
